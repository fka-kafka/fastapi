FastAPI Capstone

Get Started

This API serves as a foundational platform(backend) for a social media application. It has been developed using FastAPI as part of a project-driven approach to learning.

Key Features:

  • User Management: Create and manage user accounts with authentication.
  • Post Creation and Interaction: Users can create, read, vote on, and delete posts.
  • Authentication: Robust authentication system with a one-minute session limit.

Important Note:

This API does not have a user interface. To interact with it, please use the provided documentation over at the /docs route to access interactive documentation. This will provide detailed information on available endpoints, request/response formats, and examples.
